# Product Overview

**Precious Vault** is a full-stack precious metals trading and vault storage platform.

## Core Features

- **Trading**: Buy, sell, and convert precious metals (gold, silver, platinum, palladium)
- **Vault Storage**: Global vault locations for secure physical storage
- **Physical Delivery**: Request and track shipments of physical metals
- **Real-time Updates**: Live price feeds and portfolio tracking via WebSockets
- **KYC/Authentication**: JWT-based authentication with identity verification

## User Flow

1. Register and complete KYC verification
2. Buy precious metals (stored in vaults)
3. View portfolio and real-time prices
4. Convert between metals or sell holdings
5. Request physical delivery with tracking

## Business Model

Platform for trading and storing precious metals with options for physical delivery. Users can hold metals in secure vaults or request shipment to their address.
